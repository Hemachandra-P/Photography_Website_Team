A simple and elegant photography portfolio website built using HTML, CSS, Python, and Django to showcase stunning images in a clean, responsive layout.

🚀 Features
📷 Beautiful gallery to display photography work
🌐 Fully responsive design for all devices
🎨 Smooth hover effects and animations
🔐 Backend integration using Django for dynamic content management
⚡ Fast and efficient performance

🛠 Tech Stack
Frontend: HTML, CSS (Flexbox & Grid for layout)
Backend: Python, Django
Deployment: GitHub Pages / Netlify / Django Server
